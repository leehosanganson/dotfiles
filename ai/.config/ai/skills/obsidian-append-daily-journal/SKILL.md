# Obsidian Append Daily Journal

This skill finds the current day and appends a record of work or achievement to the daily journal.

## When to Use this Skill

- When being asked to add a record of work or achievement to the daily journal.
- When the user want to record a win or achievement for the day.

## Instructions

1. Understand the context of the work or achievement.
    - If the user request does not provide enough information about why this should be quantified as a win,
        - Ask the user to provide more information.
    - If the user cannot provide enough information,
        - Mark this work as a link (e.g. [[<work-name>]]) and wait to be used in the future as an Obsidian Link.
2. Locate the daily journal .md in the `daily-journal` directory.
3. Read the daily journal and capture the record of work or achievement.
4. Review the entry and group the work or achievement together using the Obsidian Link e.g. [[<work-name>]].

## Example

EXAMPLE 1 `USER: Record: I just finished a meeting with the team and we agreed on a new project plan.`
